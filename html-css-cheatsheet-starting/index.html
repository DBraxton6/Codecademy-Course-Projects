HTML Notes

Table Tags

Tag Name Description
<table> Table Wrapper element for HTML tables
<thead> Table Head Rows defining column headers in a table
<tbody> Table Body Rows containing the actual table data
<tr> Table Row Table row container
<td> Table Data Table data container
<tfoot> Table Foot Rows defining the footer in a table
<colspan> Column Span Defines how many columns a td element should span
<rowspan> Row Span Defines how many rows a td element should span

Lists

Tag Name Description
<ul> Unordered List Wrapper element for a bulleted (un-numbered) list.

An unordered list will look like this:

  • Milk
  • Eggs
  • Bread
<ol> Ordered List Wrapper element for a ordered (numbered) list.

An ordered list will look like this:

  1. Preheat oven
  2. Chop vegetables
  3. Bake for 30 mins
<li> List Item Wrapper element for a list item. Will either be bulleted or numbered.

Media Tags

Tag Name Description
<img> Image Element to add image media. Proper formating is <img src="imagePath" alt="description"/>
<video> Video Element to add video media. Proper formating is <video src="videoPath" controls>No video support </video>
<audio> Audio Element to add audio media. Proper formating is <audio controls>
<source src="audioPath" type="audio/type">No audio support

CSS Notes

Misc.

Note Description
tag type {...} Type the name of the tag you want to add styling to followed by opening and closing curly braces. Styling will go between the braces.
.className {...} To style a tag by its class name. More specific than tag type, so styling here will override a more general selector.
#idName {...} To style a tag by its id. More specific than tag type AND class name, so styling here will override both selectors.
!important Will prioritize a style over other declarations.
font-family: ...; Styler to change typeface
font-weight: ...; Styler to change weight of the font. Can be bold, normal, etc.
font-size: ...; Styler to change the size of the text.
text-align: ...; Styler to change the alignment of the text. Center, right, left, etc.
color: ...; Styler to change the color of the selected element.
background-color: ...; Styler to change the background color of the selected element.
background-image: ...; Styler to set the background image of the selected element.
